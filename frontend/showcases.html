Showcase #1

Image compression or static site?

A client recently had a task where he could spent limited energy on. So it was a trade off between:

  • Optimizing images to WebP
  • Moving Site from Wordpress to static

In this showcase we explore the potential benefit from moving all of his 245 images from PNG to webP, which would save CO2 because of the saved network traffic. Versus the saved CO2 from the server compute when not having to process the PHP, Database etc. for every request.

This showcase is very thourough, as it explores the complete lifecycle of the application including its hosting.
